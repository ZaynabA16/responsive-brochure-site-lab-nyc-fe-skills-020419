What I've learned in Intro to Front End Development

I've learned quite a bit in this class. We've covered HTML tags, attributes, and values. We've covered CSS fundamentals, selectors and more. We went through Flex-box and a new concept aptly named grid. And of course, we covered much, much more.

To go into some more specifics, we learned Media queries and Command terminal prompts, surge applications and iframes. We even learned about references we can use later on, things like Cheeseburger ipsum, hipster ipsum, Stumble Upon, and Bada55.io. Well, there was more but I've decided to keep it short. Just a snapshot into what we've gone over and what I've learned.

Well, I did want to mention what we'll be learning in the weeks to come. It seems like we'll cover javascript and the way it ties into CSS and HTML.

So, to summarize. We've covered:

  • Flex-box
  • Grid
  • iframes
  • ipsums
  • Terminal Commands
  • Servers and Clients
  • HTML tags and attributes
  • CSS Selectors and Rules
  • & Media Queries

And we'll cover javascript in the weeks to come!
